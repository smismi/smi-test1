Тестовое задание:


Тестовое задание делится на две части - html-вёрстку и javascript.
По срокам - каких-то жёстких ограничений нет, единственная просьба честно отписаться,
когда задание было начато и сколько чистого времени на него ушло.

По формату - зазипованный файл с одной html-ной страницей, одной js-ной и одной css-ной.

Соответствующий psd-файл, из которого можно брать графические элементы  прилагается.

HTML  вёрстка
            Cверстайте блок кнопок (1) с учётом того, что кнопку можно выбрать (см. часть задания про javascript)
            блок (2) - вставьте картинкой (и график и разлиновку)
            блок (3) - с маленьким графиком и вертикальной разлиновкой  - вставьте картинкой.
            сверстайте блок (4) с учётом того, что его можно двигать (таская за полосатую часть) и растягивать - таская за круглые бегунки
сверстайте блок (5) с учётом, что секций может быть произвольное количество
при этом области (6) между датами (то есть логически разделённые вертикальными линиями) должны кликаться.

                требования по вёрстке - IE7+, все последние браузеры большой четвёрки, максимальное использование css3

                javascript -
            кнопки должны нажиматься, нажата может быть только одна кнопка.
            по клику на кнопку должен всплывать алерт  - (нажат "День", нажата "Неделя", нажато "Увеличить во весь экран" и тд)
            бегунок (6) можно таскать и ресайзить.
по окончанию передвижения или ресайза должно всплывать сообщение о текущем состоянии бегунка (позиция, длина
и что ещё в голову прийдёт)

            Задание должно быть выполнено на чистом Javascript, без сторонних библиотек.

            и удачи!
